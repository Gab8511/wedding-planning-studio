# Roadmap

This roadmap tracks the next product-quality upgrades for VowSuite Wedding Planning Studio.

## Near Term

- Add README screenshots and a polished visual preview gallery.
- Add a printable wedding-day binder export.
- Add an in-app version and release notes panel.
- Add backup reminders before major reset or data-clearing actions.
- Improve mobile polish for wedding-day mode.
- Expand vendor contract, insurance, and payment tracking.

## Product Depth

- Add richer onboarding for a new couple.
- Add guided templates for different wedding types and planning timelines.
- Add advanced report filters for guests, vendors, budget, tasks, and timeline views.
- Add stronger accessibility support for keyboard-only and screen-reader workflows.
- Add a sample demo wedding plan that can be restored after resets.

## Collaboration

- Add shareable client-facing planning packets.
- Add clearer vendor handoff packets.
- Add role-based portal concepts for couple, planner, vendor, and day-of coordinator views.
- Explore optional cloud sync once privacy, account, and data ownership requirements are defined.

## Repository

- Keep the changelog current with each shipped upgrade.
- Add issue templates for bugs and feature requests.
- Add versioned GitHub releases for major milestones.
- Add deployment notes for GitHub Pages and custom domain setup.
