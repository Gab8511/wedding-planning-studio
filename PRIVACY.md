# Privacy

VowSuite Wedding Planning Studio is currently a static browser app. It does not require an account, server, database, or third-party analytics service.

## Local Storage

Wedding plans are stored in the browser's local storage on the device where the app is used. This can include guest details, RSVP notes, vendor information, budgets, task notes, timeline details, and planning preferences entered into the app.

## Data Sharing

The app does not automatically send wedding data to a remote server. Data only leaves the browser when a user chooses to export, copy, print, download, or otherwise share it.

## Backups

Use the app's export tools to create backups before clearing browser data, changing devices, or resetting the app for a new couple.

## Public Demo

Avoid entering sensitive real-world personal, financial, or contract information into a public demo environment unless you understand where the browser stores that data and how to clear it.

## Future Services

If hosted accounts, cloud sync, email integrations, payments, or vendor portals are added later, this privacy file should be updated before those features are released.
