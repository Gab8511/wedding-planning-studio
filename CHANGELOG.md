# Changelog

All notable changes to VowSuite Wedding Planning Studio are documented here.

## 1.1.0 - 2026-07-05

- Added real README screenshots for the dashboard, wedding-day mode, and About/Version panel.
- Added an in-app About/Version panel with storage and deployment details.
- Added printable wedding-day binder export from Day Mode and Reports.
- Added backup prompts before restoring demo data or starting a new couple.
- Added a clearer one-click sample demo restore flow.
- Added a favicon for cleaner browser and audit behavior.

## 1.0.0 - 2026-07-05

- Published the first public GitHub Pages version of the wedding planning studio.
- Added the advanced planning workspace with dashboard, guests, vendors, budget, timelines, seating, reports, portals, and studio management.
- Added local browser storage with import/export support for moving wedding plans between devices.
- Added repository metadata, live app link, and discoverable GitHub topics.
- Added project documentation for running and deploying the static app.
- Added license, privacy note, roadmap, issue templates, README badges, and preview documentation structure.
